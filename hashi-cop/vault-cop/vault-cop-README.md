#### Install Vault
#### Create a file named helm-vault-raft-values.yml with the following contents:
```
cat > helm-vault-raft-values.yml <<EOF
server:
  affinity: ""
  ha:
    enabled: true
    raft: 
      enabled: true
ui:
  enabled: true
  serviceType: "LoadBalancer"
injector:
  enabled: true
EOF
```
#### Install Vault
#### This creates three Vault server instances with an Integrated Storage (Raft) backend.
```
kubectl create ns vault
helm install vault hashicorp/vault --values helm-vault-raft-values.yml -n vault
```
